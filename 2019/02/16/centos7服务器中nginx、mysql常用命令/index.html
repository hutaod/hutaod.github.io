mysql常用命令

启动 systemctl start mysqld
停止 systemctl stop mysqld
重启 systemctl restart mysqld

nginx常用命令

启动 systemctl start nginx
停止 systemctl stop nginx
重启 systemctl restart nginx

在/etc/nginx/conf.d目录，测试nginx是否正确 nginx -t
